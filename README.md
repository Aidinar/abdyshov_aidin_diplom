Math Sense — семантическая разметка математических PDF-документов
Math Sense — веб-приложение для автоматического выделения математических терминов в PDF и тексте, их аннотирования с помощью онтологии OntoMathPRO и визуализации связей между документами. Проект включает удобный редактор, сохранение результатов в PostgreSQL и рекомендательную систему.

Возможности

Загрузка PDF‑файлов с извлечением текста (работает прямо в браузере через pdf.js)

Анализ текста на наличие математических понятий из онтологии OntoMathPRO

Подсветка найденных терминов в тексте с всплывающими определениями

Сохранение выделенных терминов в базу данных (PostgreSQL)

Автоматическое построение графа связей между файлами на основе общих терминов (коэффициент Жаккара)

Рекомендации: по заданному списку терминов система предлагает релевантные документы из БД

Просмотр истории загруженных файлов и статистики

Тёмная тема интерфейса

Интеграция с локальной LLM (Ollama) для ответов на вопросы на естественном языке

Установка и запуск
1. Клонирование репозитория


git clone <url-репозитория>
cd mathsense-backend

2. Установка зависимостей сервера

npm install

3. Настройка базы данных PostgreSQL

Убедитесь, что служба PostgreSQL запущена.

Создайте базу данных и таблицы с помощью скрипта:

node setup-db.js
Скрипт создаст базу mathsense (если её нет) и все необходимые таблицы (files, terms, file_connections).

4. Настройка переменных окружения
Скопируйте файл .env.example в .env (если файла примера нет, создайте его самостоятельно):

cp .env.example .env


5. Запуск GraphDB с онтологией OntoMathPRO
Скачайте GraphDB с официального сайта: https://www.ontotext.com/products/graphdb/

Установите и запустите GraphDB Workbench (обычно доступен на http://localhost:7200).

Создайте новый репозиторий с именем mathsense-ontology2 (тип репозитория – "GraphDB Repository").

Загрузите в репозиторий файл онтологии OntoMathPRO.ttl (должен находиться в корне проекта).

Перейдите в раздел Import → Upload RDF files.

Выберите файл OntoMathPRO.ttl и запустите импорт.

Проверьте, что репозиторий доступен по URL: http://localhost:7200/repositories/mathsense-ontology2.

6. Запуск Ollama (локальная LLM)
Скачайте Ollama с официального сайта и установите.

Запустите службу Ollama (на Windows/Mac она обычно стартует автоматически; на Linux выполните ollama serve).

Загрузите модель llama3.2:1b:

7. Запуск сервера


npm run dev

Сервер запустится на порту, указанном в .env (по умолчанию 3001). Вы должны увидеть сообщение:
Сервер запущен на http://localhost:3001
PostgreSQL подключена


8. Запуск клиентской части
Клиентская часть не требует сборки. Достаточно открыть файл index2.html в браузере любым удобным способом:

Простой способ: дважды кликнуть по файлу index2.html в проводнике.

Через локальный веб-сервер (рекомендуется для работы с некоторыми API, не требующими CORS, но в проекте CORS уже настроен). Можно использовать  live-server:

npx live-server


Затем перейдите по адресу, который покажет live-server (обычно http://127.0.0.1:8080).

Убедитесь, что клиент может обращаться к серверу на http://localhost:3001. При необходимости измените константу API_URL в начале файла script.js.

Использование системы
Загрузка PDF – перетащите файл в область загрузки или нажмите «Выбрать PDF». После обработки текст отобразится с подсвеченными терминами.

Редактор текста – введите свой текст вручную и нажмите «Анализировать текст».

Список терминов – клик на термине в тексте прокручивает страницу к соответствующему элементу списка и наоборот.

Рекомендации – после анализа система предложит другие документы с общими терминами.

LLM-помощник – на вкладке «Редактор текста» задайте вопрос на естественном языке, например: «Что такое интеграл?».
